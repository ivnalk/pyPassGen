# pyPassGen
GUI para pwgen usando un archivo y frase como semilla para contraseñas seguras que puedan ser recreadas.

# Requerimientos
* pwgen
* python3

# Ejecutar
Puedes ejecutar directamente el binario en /dist/pyPassGen.bin
o 
python3 main.py
